# lucianomanenti.com

Sito web personale di Luciano Manenti con giochi di carte online, metronomo e esercizi per pianoforte.

## Contenuti

- **Scala 40**: Gioco di carte Scala 40 online
- **Machiavelli**: Gioco di carte Machiavelli online
- **Spider Solitaire**: Gioco Spider Solitaire
- **Metronomo**: Metronomo online interattivo
- **Hanon Exercises**: Esercizi per pianoforte con file MIDI

## Tecnologie

- HTML5
- CSS3
- JavaScript (jQuery)
- MIDI (JZZ.Midi.js)

## Struttura del progetto

```
lucianomanenti.com/
├── index.html              # Homepage
├── scala40V1.html          # Gioco Scala 40
├── machiavelli.html        # Gioco Machiavelli
├── spider.html             # Spider Solitaire
├── metronome.html          # Metronomo
├── hanon.html              # Esercizi Hanon
├── complete/               # File MIDI esercizi Hanon
├── images/                 # Immagini e risorse grafiche
├── sounds/                 # File audio
└── cronotermostato/        # Applicazione cronotermostato
```

## Setup

1. Clona il repository:
```bash
git clone https://github.com/TUO_USERNAME/lucianomanenti.com.git
```

2. Apri `index.html` nel browser per visualizzare il sito.

## Note

Il sito è completamente client-side e non richiede un server per funzionare (tranne per alcune funzionalità PHP opzionali).
